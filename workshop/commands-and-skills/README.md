# Commands & Skills — Training Module

Hands-on materials for the training plan in [`../../docs/commands-and-skills-plan.md`](../../docs/commands-and-skills-plan.md).

**Intermediate module** — **prerequisite:** you must have completed Phases 1–5 of the TodoMVC or Juice Shop workshop and have a running target checkout with `CLAUDE.md`, hardened `settings.json`, and the `block-secrets.sh` hook in place. Phases 1–5 here are prerequisite checks, not setup walkthroughs.

## Where to work

This folder is reference material. Run Claude Code in your **existing** TodoMVC or Juice Shop checkout (the one you used for the primary workshop) and port commands/Skills into that repo's `.claude/` folder. See [`../../docs/commands-and-skills-plan.md`](../../docs/commands-and-skills-plan.md) for the full layout.

## How to Use

1. Work through Phases 1–4 in order (they're short — mostly prerequisite checks + one decision-framework drill).
2. Phase 5: confirm your target repo(s). Pick which tracks you'll do.
3. Phase 6: do the Track(s) that match your target(s).
4. Phase 7: iterate until the command/Skill is durable. Commit it.
5. Phase 8: reflect on when each tool shape wins.
