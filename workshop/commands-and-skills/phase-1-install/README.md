# Phase 1 — Confirm Setup (prerequisite check)

**Time:** 5 min
**Goal:** verify Claude Code is ready. **No install walkthrough.**

> This workshop is **not self-contained**. If you've never installed Claude Code, stop here and do Phase 1 of the [TodoMVC workshop](../../todomvc/phase-1-install/README.md) or [Juice Shop workshop](../../juice-shop/phase-1-install/README.md) first — they walk through install, authentication, and the UI tour.

## Check

- [ ] `claude --version` runs
- [ ] `/login` succeeds in a fresh session
- [ ] `/help` lists the built-in commands (`/review`, `/security-review`, `/init`, etc.)
- [ ] `shift+tab` cycles default → accept-edits → plan
- [ ] You're comfortable with `TodoWrite` and the `#` quick-memory shortcut (both covered in the primary workshop's Phase 1)

## If any checkbox is unchecked

Go back to the primary workshop. This module assumes you already have install, auth, and basic UI fluency in muscle memory.

## No before/after

Prerequisite check only.
