# Phase 8 — Reflection

**Time:** 10 min

## Questions

1. **Decision accuracy** — of your command and Skill, which one more often regretted being made into infrastructure instead of staying a plain prompt?
2. **Reaching** — did Claude reach for your Skill when you expected it to? Where did it miss? What phrasing in your `description` didn't match real user wording?
3. **Guardrails** — did `allowed-tools` catch something the command otherwise would have done? Try **removing** the list and re-running on a safe input — see what Claude reaches for.
4. **Borrowed patterns** — which patterns from [`addyosmani/agent-skills`](https://github.com/addyosmani/agent-skills) showed up in your Skill? Which did you skip as unnecessary?
5. **Tipping point** — at what team size does it stop being worth committing custom commands vs. just teaching the prompt? (Consider who maintains them.)

Use `#` to append findings about what makes a command/Skill durable vs. fragile — this is the most portable learning from the workshop.
