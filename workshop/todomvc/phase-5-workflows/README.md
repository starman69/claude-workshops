# Phase 5 — Core Workflows

**Time:** 45 min
**Goal:** Fluency with the tools you'll use every day.

## Exercises (in `exercises/`)

Work through each with Claude. For each, note **which tool Claude chose** and **why**.

1. **Find a function** — `Grep`, not `Bash(grep)`. See `01-find-function.md`.
2. **Plan a change** — plan mode, review, exit to execute. See `02-plan-mode.md`.
3. **Live UI inspection** — Chrome DevTools / Playwright MCP. See `03-browser-mcp.md` and `../../../docs/mcp-setup.md`.
4. **Context & session hygiene** — `/clear`, `/compact`, when to restart. See `04-context-hygiene.md`.
5. **Prompting craft** — rewriting vague prompts with anchors + scope + done condition. See `05-prompting-craft.md`.

## Run these slash commands live

Don't just read about them — actually run each in a throwaway session:

- `/help` — what's available
- `/model` — confirm the active model
- `/init` — on a tiny throwaway repo, see what it generates
- `/review` — on a one-file edit, read the output critically
- `/security-review` — same, with a security lens

## No before/after

These are live exercises. The "after" is fluency, not a file.
